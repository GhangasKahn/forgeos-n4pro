ForgeOS eyes — Nothing Phone setup

Scan the QR codes below. Same Wi‑Fi as your Mac/printer (192.168.1.x).

1. Install IP Webcam

Scan → Google Play → IP Webcam (Pavel Khlebovich)

https://play.google.com/store/apps/details?id=com.pas.webcam

2. Start the server

  1. Open IP Webcam
  2. Scroll to bottom → Start server
  3. Note URL like http://192.168.1.XX:8080
  4. Point phone at the print bed (~30–45° angle)

3. Full instructions (GitHub)

https://github.com/GhangasKahn/forgeos-n4pro/blob/master/docs/PHONE_AS_CAMERA.md

4. Tell Grok / run on Mac

After server is running, on the Mac:

python3 scripts/phone_cam_test.py http://PHONE_IP:8080
python3 -m forgeos.vision.service --phone-url http://PHONE_IP:8080 --interval 0.5 -v

Or reply in chat with your phone URL.
