# Hardware upgrades

## Required for plan goals
1. Hardened nozzle (HTPLA-CF: ≥0.5 mm preferred)
2. Clean PTFE / feed path
3. ADXL345 **or** Beacon/Eddy

## Shop tools
- Calipers / 100 mm reference
- Filament dryer
- Controlled anneal oven
- Scale

## Install notes
- ADXL: enable sections in `klipper/base/printer_n4pro.cfg`
- Eddy/Beacon: follow vendor Klipper module; journal shaper freqs after each `SHAPER_CALIBRATE`
