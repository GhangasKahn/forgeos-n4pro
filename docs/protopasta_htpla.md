# Protopasta HTPLA / HTPLA-CF — shop protocol

## HTPLA
- Print hotter than standard PLA for fusion
- Anneal ~110 °C, ~60 min (verify with your oven; use oven thermometer)
- Expect XY shrink / Z growth — use ForgeOS scale seeds then learn from coupons
- Good for jigs that see mild heat or need stiffness after anneal

## HTPLA-CF
- **Hardened nozzle**, prefer ≥0.5–0.6 mm
- Lower max volumetric flow
- More brittle filament path — gentle dry-box feed
- Anneal shrink typically less aggressive than pure HTPLA but still measure

## Never
- Soft brass + CF
- Skip drying for precision fixtures
- Trust first anneal scale without measuring a coupon
